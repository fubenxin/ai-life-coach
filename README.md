# AI生活教练网站

## 项目概述
这个项目是一个基于千问大模型API的AI生活教练网站，用户可以通过与AI助手进行对话，获取生活建议和成长指导。网站采用简洁友好的界面设计，提供流畅的交互体验。

## 功能特性
- 实时与AI进行对话交流
- 响应式设计，适配各种设备屏幕
- 聊天记录的展示和管理
- 现代化UI设计，提供友好的用户体验
- 流式输出，提升对话流畅感
- 作为生活教练，提供个性化的成长建议

## 项目结构

```
├── index.html         # 主页面，包含聊天界面
├── css/               # 样式文件夹
│   └── styles.css     # 主样式文件
├── js/                # JavaScript文件夹
│   └── script.js      # 前端交互逻辑
├── server/            # 后端服务器
│   └── server.js      # Node.js服务器，处理API请求
└── package.json       # 项目依赖配置
```

## 技术栈
- **前端**：HTML5, CSS3, JavaScript
- **后端**：Node.js, Express
- **API**：千问大模型API

## 安装和运行指南

### 前置条件
- 已安装Node.js环境（版本14.x或更高）
- 已获取千问大模型API密钥（本项目已在server.js中配置）

### 安装步骤
1. 克隆或下载项目文件到本地
2. 打开命令行，进入项目根目录
3. 安装所需依赖：
   ```
   npm install
   ```
4. 启动后端服务器：
   ```
   node server/server.js
   ```
   或者使用开发模式（自动重启）：
   ```
   npm run dev
   ```
5. 服务器启动后，会在控制台显示：
   ```
   服务器运行在 http://localhost:3000
   健康检查: http://localhost:3000/api/health
   聊天API: http://localhost:3000/api/chat
   ```
6. 在浏览器中访问 http://localhost:3000 或者打开index.html文件

### 故障排除
- 如果API请求失败，请检查网络连接和API密钥是否有效
- 如果服务器无法启动，请确保端口3000未被占用，或修改server.js中的PORT配置
- 如果前端无法连接到后端，请确认CORS设置是否正确
- 对于流式输出问题，可以检查浏览器控制台是否有相关错误信息

## 界面说明
网站界面采用简洁现代的设计风格，主要包含以下部分：

1. **头部区域**：显示网站标题和简短说明
2. **聊天区域**：展示用户与AI助手的对话记录
3. **输入区域**：用户输入问题或对话内容的文本框和发送按钮

## 使用说明
1. 在输入框中输入您的问题或想法
2. 点击发送按钮或按Enter键提交
3. 等待AI助手的回复，回复将以流式方式显示
4. 继续对话，AI助手将根据上下文提供相应的建议和指导

## 注意事项
- 请妥善保管API密钥，避免泄露
- API请求可能受到速率限制，请合理使用
- 响应时间可能受到网络状况和服务器负载影响
